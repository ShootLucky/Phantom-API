Phantom

Lua API Documentation

Introdução

Phantom proporciona um poderoso mecanismo de scripts Lua com uma API completa para:

  • Manipulação de objetos de jogo através do sistema de instâncias
  • Desenho de baixo nível com formas, texto e linhas
  • Widgets UI interativos e menus
  • Serviços para players, workspace, lighting e muito mais

Lua Virtual Machine

Gerenciamento do motor Lua e execução de scripts.

bool initialize() VM

Inicializa a Lua VM. Deve ser chamado antes de executar qualquer script.

Retorna: true se bem-sucedido

if (lua_vm::initialize()) {
    // VM está pronta
}
bool is_ready() VM

Verifica se a Lua VM está inicializada e pronta.

Retorna: true se pronta

bool execute_string(const std::string& script, const std::string& chunk_name = "lua_script") VM

Executa código Lua de uma string.

Parâmetros: script (código Lua), chunk_name (nome para debug)

lua_vm::execute_string("print('Hello, Lua!')");
lua_vm::execute_string("x = 42", "meu_chunk");
bool execute_file(const std::filesystem::path& file_path) VM

Executa código Lua de um arquivo.

Parâmetros: file_path (caminho para arquivo .lua)

void on_frame(float delta_time) VM

Chamado a cada frame. Coloque isto no seu game loop.

while (jogo_rodando) {
    float delta = calcular_delta_time();
    lua_vm::on_frame(delta);
    renderizar_frame();
}

Sistema de Instâncias

Modelo de objeto hierárquico para organizar objetos do jogo. Todos os objetos herdam de instance.

Propriedades Básicas

std::string get_name() const Instance

Obtém o nome da instância.

local player = workspace:FindFirstChild("Player1")
print(player:GetName())  -- Output: Player1
void set_name(const std::string& name) Instance

Define o nome da instância.

std::string get_class_name() const Instance

Obtém o nome da classe.

Relacionamentos Parent-Child

std::vector<std::shared_ptr<instance>> get_children() const Instance

Obtém todos os filhos diretos.

local children = workspace:GetChildren()
for _, child in ipairs(children) do
    print(child:GetName())
end
std::shared_ptr<instance> find_first_child(const std::string& name) const Instance

Encontra o primeiro filho direto por nome.

local player = workspace:FindFirstChild("Player1")
std::shared_ptr<instance> find_first_descendant(const std::string& name) const Instance

Encontra o primeiro descendente em qualquer profundidade.

Atributos

void set_attribute(const std::string& key, const attribute_value& value) Instance

Define um atributo. Tipos: bool, double, string, null.

player:SetAttribute("health", 100.0)
player:SetAttribute("team", "red")
player:SetAttribute("active", true)
std::optional<attribute_value> get_attribute(const std::string& key) const Instance

Obtém um valor de atributo.

local valor = player:GetAttribute("health")
if valor then
    print("Saúde: " .. valor)
end

Drawing API

Primitivos de desenho de baixo nível para renderizar na tela.

std::shared_ptr<drawing_object> create_object(const std::string& type_name) Drawing

Cria um objeto desenhável.

Tipos: "square", "text", "line"

-- Criar um quadrado
local box = drawing.create_object("square")
box.position.x = 100
box.position.y = 100
box.size.x = 50
box.size.y = 50
box.color = {r=1, g=0, b=0, a=1}
box.filled = true
std::shared_ptr<drawing_object> find_object(int object_id) Drawing

Encontra um objeto desenhável por ID.

bool remove_object(int object_id) Drawing

Remove um objeto desenhável.

void render_all() Drawing

Renderiza todos os objetos visíveis.

void clear_all() Drawing

Remove todos os objetos desenhados.

Propriedades de Objects

position

Coordenadas X, Y

size

Largura e altura

color

Cor RGBA (0-1)

thickness

Espessura de linha

filled

Preencher forma

z_index

Ordem de desenho

UI Widget System

Crie elementos de interface interativos.

Gerenciamento de Abas

int create_tab(const std::string& name) UI

Cria uma nova aba de UI.

local tab_id = ui.create_tab("Configurações")
print("Aba criada: " .. tab_id)
bool remove_tab(int tab_id) UI

Remove uma aba e todos seus widgets.

std::vector<tab_descriptor> get_tabs_snapshot() UI

Obtém todas as abas atuais.

Tipos de Widgets

Checkbox

Toggle booleano

add_checkbox_widget(tab, key, label, default)

Slider

Deslizador numérico

add_slider_widget(tab, key, label, min, max, default)

Dropdown

Seleção única

add_dropdown_widget(tab, key, label, options, index)

Button

Botão clicável

add_button_widget(tab, key, label, callback)

ColorPicker

Seleção de cor

add_colorpicker_widget(tab, key, label, color)

InputText

Campo de texto

add_input_text_widget(tab, key, label, default)

Valores de Widget

bool get_widget_value(int tab_id, const std::string& key, widget_value& out_value) UI

Obtém o valor atual de um widget.

bool set_widget_value(int tab_id, const std::string& key, const widget_value& in_value) UI

Define o valor de um widget.

Services

Serviços globais fornecendo acesso aos principais sistemas do jogo.

Players Service

Acesso ao gerenciamento de players e informações.

game:GetService("Players")

Workspace Service

Acesso ao mundo do jogo e ambiente.

game:GetService("Workspace")

RunService

Eventos baseados em frames e timing.

game:GetService("RunService")

Lighting Service

Controle de iluminação e efeitos visuais.

game:GetService("Lighting")

Exemplos de Código

Criar Painel de Configurações

-- Criar aba de configurações
local tab_id = ui.create_tab("ui example")
ui.clear_tab(tab_id)

ui.checkbox(tab_id, "enabled", "Enabled", true)
ui.colorpicker(tab_id, "accent", "Accent", { 0.15, 0.7, 1.0, 1.0 })

ui.checkbox(tab_id, "show_boxes", "Show Boxes", false)
ui.colorpicker(tab_id, "outline", "Outline", { 1.0, 1.0, 1.0, 1.0 })

ui.checkbox(tab_id, "hold_to_aim", "Hold To Aim", true)
ui.keybind(tab_id, "aim_key", "Aim Key", 0x02, 0)

ui.checkbox(tab_id, "toggle_menu", "Toggle Menu", true)
ui.keybind(tab_id, "menu_key", "Menu Key", 0x2D, 1)

ui.dropdown(tab_id, "mode", "Mode", { "Closest", "Fov", "Distance" }, 0)
ui.dropdown(tab_id, "hitpart", "Hitpart", { "Head", "Torso", "HumanoidRootPart" }, 0)

ui.slider(tab_id, "fov", "Fov", 25, 500, 120)
ui.slider(tab_id, "smoothness", "Smoothness", 1, 100, 20)

Desenhar na Tela

-- Criar caixa vermelha
local box = Drawing:CreateObject("square")
box.position = {x = 100, y = 100}
box.size = {x = 150, y = 100}
box.color = {r = 1, g = 0, b = 0, a = 1}
box.thickness = 2
box.filled = false

-- Criar texto
local text = Drawing:CreateObject("text")
text.position = {x = 110, y = 110}
text.text = "PLAYER 1"
text.text_size = 16
text.color = {r = 1, g = 1, b = 1, a = 1}

-- Renderizar
Drawing:RenderAll()

Trabalhar com Instâncias

-- Obter workspace
local workspace = game:GetService("Workspace")

-- Encontrar player
local player = workspace:FindFirstChild("Player1")
if player then
    print("Encontrado: " .. player:GetName())

    -- Definir atributos
    player:SetAttribute("health", 100)
    player:SetAttribute("team", "red")

    -- Obter todos os filhos
    local children = player:GetChildren()
    for _, child in ipairs(children) do
        print("Filho: " .. child:GetName())
    end
end
